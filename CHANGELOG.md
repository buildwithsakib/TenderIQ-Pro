# Changelog

All notable changes to **TenderIQ Pro** are documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added
- Part 2: full Settings editor persisted in config.json (Save / Restore
  Default / Reset), SQLite-backed History window with search, delete,
  CSV export and open-report, config-driven dashboard sync, persisted
  theme preference, and first-run guided welcome dialog.
- Part 1: PySide6 application architecture, main window with sidebar
  navigation, complete dashboard layout, dark and light themes, SVG icon
  set, application metadata module, and backend integration plan.

## [1.0.0-backend] - 2026-07-03

### Added
- Working GeM tender scanner backend: Playwright browser automation,
  PDF download and field extraction, eligibility matching, duplicate bid
  tracking, incremental Excel reporting, and central logging.
