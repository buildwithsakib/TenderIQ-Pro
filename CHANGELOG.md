# Changelog

All notable changes to **TenderIQ Pro** are documented in this file.

## Version 1.0.0 (04 July 2026)

### Added
- Modern PySide6 desktop interface.
- Automated GeM bid scanning using Playwright.
- PDF download and tender information extraction.
- Keyword-based tender matching.
- Configurable validation rules and filters.
- Incremental Excel report generation.
- Duplicate bid detection and tracking.
- SQLite-based scan history.
- Dark and Light theme support.
- Settings management with `config.json`.
- Welcome dialog for first-time users.
- Integrated logging and error reporting.

### Changed
- Improved scanner stability and browser restart handling.
- Enhanced PDF download reliability.
- Optimized dashboard statistics and UI responsiveness.

### Fixed
- Fixed duplicate bid tracking issues.
- Fixed PDF cleanup after processing.
- Fixed report save fallback handling.
- Fixed various UI and stability improvements.
