# Install with:
#   pip install -r requirements.txt
#
# After installing, you MUST also install the Playwright browser binaries:
#   playwright install chromium
#
# NOTE ON PyMuPDF VERSION:
# PyMuPDF only ships pre-built wheels for the Python versions that were
# officially supported at the time of that release. Pinning too old a
# version (e.g. 1.24.x, released in 2024) means pip cannot find a wheel on
# newer Python installs (3.13/3.14) and instead tries to compile from
# source, which requires a full Visual Studio C++ toolchain and fails on
# most machines. 1.26.11+ officially ships wheels for Python 3.10-3.14, so
# a lower-bound constraint (not an exact pin) is used here to always pull a
# version with a matching pre-built wheel.
#
# NOTE ON PLAYWRIGHT VERSION:
# Older Playwright releases (<=1.47.0) internally hard-pin an exact,
# outdated version of their "greenlet" dependency (greenlet==3.0.3), which
# has no pre-built wheel for Python 3.13/3.14 and fails to compile without
# Visual Studio installed - this was a known upstream issue. Versions
# 1.55.0+ depend on a modern, flexible greenlet range that has wheels for
# every current Python version, avoiding the problem entirely.
#
##############################################################################

playwright>=1.55.0
PyMuPDF>=1.26.11
openpyxl==3.1.5
